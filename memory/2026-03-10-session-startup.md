OTP for NY State - Step process with timelines

 

1) Define the legal scope + producer obligation model (CA + NY)

What you do

Write a scope memo: what counts as “artificial turf” (sports + landscape?), what’s included (pad/infill/adhesives), who is the “producer” (brand owner/importer/private label), and what “placed on market” unit you’ll use (sq yd / kg).
Draft the compliance mechanism: registration, reporting cadence, enforcement ladder.
Timeline: 2–4 weeks

2) Form the founding coalition + secure seed funding

What you do

Recruit founding producers (top brands/importers) + installers/field builders (operational partners).
Sign a Founding MOU and collect startup assessments (or commitments).
Timeline: 4–8 weeks (can overlap with Step 1)

3) Incorporate “One Turf PRO” + stand up governance

What you do

Incorporate (often nonprofit), open bank accounts, insurance, basic corporate governance.
Establish Board + committees (Finance/Audit, Ops, Standards, Stakeholder).
Adopt key policies: conflict of interest, ethics, procurement, records, privacy.
Timeline: 3–6 weeks

4) Design the “turf materials system” and acceptance standards

What you do

Publish a standardized bill-of-materials taxonomy: fiber, backing, coatings, pad, infill types.
Define acceptance/rejection rules and contamination thresholds.
Define end-of-life pathways hierarchy (reuse, recycling, downcycling, disposal last).
Timeline: 3–6 weeks

5) Build the certification + audit framework (especially if you want “single-material” positioning)

What you do

Define product registry requirements (BOM declarations, labeling, disassembly instructions).
Create audit protocol: desk audits, sampling, processor verification, chain-of-custody rules.
Set penalties: fee uplifts, de-listing products, chargebacks for rejected loads.
Timeline: 4–8 weeks

6) Build the collection + logistics network (project-based removals)

What you do

Qualify and contract takedown contractors.
Select consolidation yards (at least 1–2 in NY and 2–3 in CA to start).
Set hauling contracts and service levels (pickup windows, minimum loads).
Create SOPs for rolling/wrapping/labeling/weighing/photo documentation.
Timeline: 8–12 weeks

7) Contract processing capacity + outlets (the make-or-break step)

What you do

Run an RFP for processors (NY and CA; include contingency processors).
Contract minimum throughput, pricing, residue allocation, and reporting format.
Secure offtake / end markets for recovered fractions (pellets/flakes).
Timeline: 8–16 weeks (often the longest lead time)

8) Set the fee model + finance controls

What you do

Choose fee basis: $/sq yd (most intuitive) and optionally eco-modulated tiers.
Build a first-year budget (ops, transport, processing, admin, outreach, reserve).
Establish reserve policy and audited financial reporting.
Timeline: 4–6 weeks (can overlap with Steps 6–7)

9) Deploy the compliance IT system (registration → reporting → invoicing → manifests)

What you do

Implement a system for producer registration, product registry, fee invoicing.
Add project-manifest workflow (pickup scheduling + chain-of-custody + weights).
Processor portal and reconciliation (mass balance, yield, residue reporting).
Audit workflow and annual report generation.
Timeline:

MVP system: 6–10 weeks
More robust/automated: 10–16 weeks
10) Stakeholder engagement + regulator-facing plan package

What you do

Build a public-facing plan: targets, network coverage, convenience, education, KPIs.
Stakeholder sessions: municipalities, schools, sports orgs, environmental groups.
Publish a “how to participate” guide for generators (schools/cities).
Timeline: 4–8 weeks (overlaps with Steps 6–9)

11) Pilot launch (prove the real unit economics)

What you do

Run pilots in 1–2 regions per state.
Track: cost/field, contamination rate, yield, % diversion, service times.
Adjust acceptance standards, contracts, and fees.
Timeline: 8–12 weeks of pilot operations

12) Full program launch + first annual reporting cycle

What you do

Statewide coverage ramp, formal enforcement begins.
