Tool policy

tools.profile

Per-agent override for the actual agent bound to that Discord session
Discord may not be hitting the agent you think it is. Bindings resolve by channel, guild, account, peer, then fall back to the default agent.

Provider-specific restriction
Even if global config allows exec, tools.byProvider can narrow the set further for the model/provider used by that agent.

Only after that does exec approval matter
Your host: gateway, security: full, ask: off settings affect how host exec runs once exec is already available. Exec approvals are “in addition to tool policy,” not a replacement for it.
