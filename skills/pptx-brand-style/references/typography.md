# Typography

## Fonts
- Headings: Archivo
- Body: Fira Sans
- Labels: Fira Mono

## Sizes
- Title slide: 36–40pt
- Section headers: 20–24pt
- Headline: 28–32pt
- Body: 18–20pt
- Small labels: 12–14pt

## Fallbacks
If fonts are not installed, use:
- Headings: Calibri Bold
- Body: Calibri
- Labels: Consolas
